# Employee-Tracker

## An application created to allow the user to view and update employees, departments, and roles via the command line.
This app uses Node, Inquirer, and MySQL.

[![License](https://img.shields.io/badge/License-BSD%202--Clause-blue.svg)](https://opensource.org/licenses/BSD-2-Clause)

## Installation
To use this app fork this repo. Next you will run ```npm install``` to install the dependencies. 

## Usage
From the command line type ```node app```.
